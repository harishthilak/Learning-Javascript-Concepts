for.....of loop

for...of is used to iterate over iterable objects like arrays and strings.

for...in Loop

for...in is used to iterate over the keys (properties) of an object.

forEach Loop (Array Method)

Definition: A higher-order function that executes a provided function once for each array element.

.map() Method

Creates a new array by applying a function to each element.

.filter() Method

Definition: Returns a new array with elements that match a condition.

Labeled Break

Labels let you break or continue out of outer loops.
